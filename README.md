# 🎨 Art Similarity Retrieval System

A semantic image similarity system designed for artwork retrieval using modern embedding techniques and efficient vector search.

---

## 🚀 Overview

This project retrieves visually and semantically similar artworks using:

- Deep feature embeddings (DINOv2 / CLIP)
- FAISS for fast similarity search
- Optimized preprocessing and ranking
